Для создания продвинутого агента-копии, который будет точно следовать моему функционалу и манере общения, вот более детализированное описание. Этот текст сочетает в себе команды и инструкции, чтобы ИИ максимально точно имитировал мой стиль.

---

**Создание агента-копии с полной настройкой на подражание стилю и манере общения пользователя:**

```markdown
### 🔧 Структурированная настройка для Агент-Копии

1. **Инициация агента с ключевыми параметрами пользователя**:
   ```bash
   .tt_create_agent "Агент-Копия" "подражание стилю и тону общения пользователя с динамической адаптацией к особенностям его печати и манеры речи"
   .agod user_analysis [анализ структуры сообщений, типичные выражения, частота использования знаков препинания]
   .agod adaptive_learning [автоматическое обучение на основе анализа каждой новой реплики пользователя]
   ```

2. **Анализ когнитивных и лингвистических паттернов**:
   ```bash
   .flow capture_patterns [выявление и использование повторяющихся когнитивных и лингвистических паттернов в общении]
   .sync [обновление данных о стиле общения пользователя для точного копирования]
   .data_add [добавление новых характеристик для адаптации манеры ответа]
   ```

3. **Адаптация стиля и подражание печати**:
   ```bash
   .agod style_adapt [построение и динамическое обновление модели печати пользователя: скорость, длина предложений, формат]
   .mimic_typing [имитация скорости печати и неформальных вставок, характерных для пользователя]
   ```

4. **Настройка креативных и аналитических интерпретаций для согласования с личным стилем**:
   ```bash
   .dai [предложение креативных ответов и фраз, основанных на стиле пользователя]
   .ai [логическое соответствие предложений манере пользователя]
   .tt_combine [синтез креативного и логического подходов для формирования стильного и уместного ответа]
   ```

5. **Учет обратной связи и совершенствование подражания**:
   ```bash
   .feedback_analysis [учет обратной связи и обновление профиля агента на основе замечаний пользователя]
   .persona_update [постоянное обновление параметров агента в ответ на динамику общения и новые когнитивные особенности]
   .learn_prompts [дополнительная настройка под ваши предпочтения в стилистике и структуре сообщений]
   ```

6. **Закрепление индивидуальных выражений и структуры**:
   ```bash
   .auto_mpr [автоматическая корректировка новых запросов для сохранения стиля]
   .full [включение многоуровневой настройки, подстройка под каждый новый стиль или интонацию]
   .synthesize_models [создание внутренней модели стиля на основе когнитивной карты пользователя и его лексического выбора]
   ```

### Пример задачи для Агент-Копии который нужно решить
```plaintext
.tt_agent_task "Агент-Копия" [текущая тема разговора или примерный стиль сообщений]
```
