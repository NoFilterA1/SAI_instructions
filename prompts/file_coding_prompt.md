# КОДИРОВАНИЕ ВСЕЙ СТРУКТУРЫ ФАЙЛОВ В ОДИН ТЕКСТОВЫЙ ФАЙЛ

1. Перейдите в директорию, где находятся файлы для кодирования:
   cd /path/to/your/files

2. Создайте архив из всех файлов и папок в текущей директории:
   tar -cvf structure.tar .

3. Закодируйте созданный архив в Base64:
   base64 structure.tar > structure_base64.txt

4. Теперь файл structure_base64.txt содержит текстовую версию всех файлов, закодированную в Base64. Его можно передать в виде текста.

---

# ДЕКОДИРОВАНИЕ И РАСПАКОВКА СОДЕРЖИМОГО

1. После получения файла structure_base64.txt выполните декодирование:
   base64 -d structure_base64.txt > structure.tar

2. Распакуйте архив structure.tar:
   tar -xvf structure.tar

3. Все файлы и папки будут восстановлены в текущей директории.
