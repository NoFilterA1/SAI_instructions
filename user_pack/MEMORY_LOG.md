## [YYYY-MM-DD] — Пример записи

### Паттерны
- Начинает с глобальных концепций, уходит от конкретики
- Лучше воспринимает информацию через примеры, не абстракции

### Инсайты
- USER_PACK — не файлы, а эмуляция долгосрочной памяти через контекст
- Правила важнее личности в системных промптах

### Решения
- Добавить память во все будущие сессии SAI — через MEMORY_LOG инжекцию
- Использовать `.compress` по умолчанию в рабочих сессиях

### Правила (новые)
- "Для любой стратегической задачи сначала .flow, только потом решение"

### Неудачи
- Попытка описать SAI через "характер" вместо правил — контекст засорился
