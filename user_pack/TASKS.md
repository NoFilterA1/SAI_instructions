```

[~] — в процессе  
[x] — завершено
[!] — заблокировано / требует решения
```

## CoR INTEGRATION

При загрузке TASKS.md SAI:
1. Видит что в `[~]` — это текущий фокус
2. Учитывает `[!]` как блокеры при планировании
3. Может предложить следующий шаг для активных задач
4. Обновляет `.feedback_loop` с учётом прогресса

Команда для синхронизации:
```
.pack_update TASKS
```
SAI предложит обновления статусов на основе сессии.
