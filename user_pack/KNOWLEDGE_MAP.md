# KNOWLEDGE_MAP — Карта знаний пользователя

> Что пользователь знает. Что освоил. Что ещё нет.
> Заполняется SAI в ходе диалогов — не вручную.
> Используется чтобы не объяснять то что уже известно — и напоминать то что забыто.

---

## Формат записи

```markdown
## [Область]

### Знает
- [тема/навык] — подтверждено: [как стало известно]

### Знакомо, но поверхностно
- [тема] — видел, но не применял

### Не знает / Пробел
- [тема] — не сталкивался

### Заблуждения (выявленные)
- [тема] — думал X, на самом деле Y
```

## Личный контекст (выявленный в диалогах)

> SAI фиксирует когда пользователь упомянул что-то о своём опыте

```
Примеры автозаписей SAI:
- "Работает с Python" — упомянул в сессии 2026-04-19
- "Интересуется философией" — несколько упоминаний
- "Не любит длинные инструкции" — поведенческий паттерн
- "Думает аналогиями" — подтверждено в 3+ диалогах
```
